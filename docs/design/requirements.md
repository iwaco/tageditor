# LoRA トレーニング用 画像タグエディター 要件定義書

## 1. 概要

### 1.1 目的

LoRA（Low-Rank Adaptation）トレーニング用データセットの画像タグを効率的に編集・管理するためのWebベースUIを提供する。一覧性の高いインターフェースにより、大量の画像に対するタグの確認・編集・一括操作を容易にする。

### 1.2 対象ユーザー

- LoRAモデルのトレーニングデータセットを準備するユーザー
- Stable Diffusion等の画像生成モデル向けファインチューニングデータを管理するユーザー

### 1.3 技術スタック（候補）

| レイヤー | 技術 | 備考 |
|---------|------|------|
| フロントエンド | React + TypeScript | SPA構成 |
| UIライブラリ | shadcn/ui + Tailwind CSS | |
| バックエンド | Node.js / Go / Python (FastAPI) | ファイルI/O中心のため軽量で可 |
| 画像配信 | ローカルファイルサーバー | サムネイル生成含む |

## 2. データモデル

### 2.1 入力データ構造

```
<ROOT_FOLDER>/
├── <category_A>/
│   ├── image_001.jpg
│   ├── image_001.txt
│   ├── image_002.jpg
│   ├── image_002.txt
│   └── ...
├── <category_B>/
│   ├── photo_001.jpg
│   ├── photo_001.txt
│   └── ...
└── ...
```

### 2.2 命名規則

| ファイル種別 | パターン | 例 |
|-------------|---------|-----|
| 画像ファイル | `${BASE_NAME}.jpg` | `sunset_beach_001.jpg` |
| タグファイル | `${BASE_NAME}.txt` | `sunset_beach_001.txt` |

- 画像とタグファイルは同一ディレクトリに配置され、拡張子のみが異なる
- 画像形式は `.jpg` を基本とするが、`.png`, `.webp` への拡張も考慮する

### 2.3 タグファイルフォーマット

```
tag1, tag2, tag with spaces,  tag3 , tag4
```

- セパレーター: `,`（カンマ）
- 各タグの前後の空白はトリムする
- タグ自体にスペースを含むことがある（例: `long hair`, `blue eyes`）
- 空タグ（カンマの連続）は無視する
- ファイルエンコーディング: UTF-8

### 2.4 内部データモデル

```typescript
interface ImageEntry {
  id: string;                  // ユニークID（パスベース）
  baseName: string;            // ファイルベース名
  category: string;            // サブフォルダー名（カテゴリ）
  imagePath: string;           // 画像ファイルの相対パス
  tagFilePath: string;         // タグファイルの相対パス
  tags: string[];              // パース済みタグ配列
  thumbnailUrl: string;        // サムネイルURL
  metadata?: ImageMetadata;    // 画像メタデータ（オプション）
}

interface ImageMetadata {
  width: number;
  height: number;
  fileSize: number;
}

interface TagStats {
  tag: string;
  count: number;               // 全画像中の使用回数
  categories: string[];        // 使用されているカテゴリ一覧
}
```

## 3. 機能要件

### 3.1 データ読み込み・書き出し

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| F-IO-01 | フォルダー読み込み | ルートフォルダーを指定し、サブフォルダー構造ごと画像・タグファイルを再帰的に読み込む | 必須 |
| F-IO-02 | タグファイルパース | カンマ区切りのタグを配列に変換。前後スペースをトリム | 必須 |
| F-IO-03 | タグファイル保存 | 編集後のタグをカンマ区切りでテキストファイルに書き戻す | 必須 |
| F-IO-04 | 自動保存 | タグ編集後に自動保存（デバウンス付き）。明示的な保存ボタンも提供 | 推奨 |
| F-IO-05 | サムネイル生成 | 一覧表示用のサムネイルを生成・キャッシュする | 必須 |
| F-IO-06 | 対応画像形式の拡張 | `.jpg` に加え `.png`, `.webp` をサポート | 推奨 |

### 3.2 画像ブラウジング

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| F-BR-01 | カテゴリツリー | サブフォルダー構造をツリー表示し、カテゴリごとに画像をフィルタリング | 必須 |
| F-BR-02 | サムネイルグリッド | 選択したカテゴリの画像をサムネイルグリッドで一覧表示 | 必須 |
| F-BR-03 | 画像プレビュー | サムネイルクリックで拡大プレビュー表示（詳細は 3.2.1 参照） | 必須 |
| F-BR-04 | ソート | ファイル名順、タグ数順、更新日時順でソート可能 | 推奨 |
| F-BR-05 | 画像選択 | 単体選択、Shift+クリックで範囲選択、Ctrl+クリックで個別追加選択 | 必須 |
| F-BR-06 | 未タグ画像フィルター | タグが0件または指定数以下の画像のみ表示するフィルター | 推奨 |
| F-BR-07 | タグ数バッジ | サムネイル上にタグ数をバッジ表示 | 推奨 |

#### 3.2.1 画像詳細ビュー

画像の細部（顔の表情、手の描写、背景ディテールなど）を確認しながらタグ編集を行うための専用ビュー。

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|

| F-DV-02 | ズーム操作 | マウスホイールまたはピンチで拡大/縮小。100%表示（ピクセル等倍）ボタンも提供 | 必須 |

| F-DV-04 | Fit to Window | ウィンドウに合わせた自動フィット表示。トグルで100%表示と切り替え | 必須 |
| F-DV-05 | タグ編集パネル併用 | 詳細ビュー中もタグクラウド・タグ編集パネルが横に表示され、即座にタグを編集可能 | 必須 |

| F-DV-07 | スライドショーモード | 自動または手動で画像を順に表示し、確認漏れを防止 | 任意 |
| F-DV-08 | 画像情報オーバーレイ | ファイル名、解像度、ファイルサイズ等のメタ情報をオーバーレイ表示（トグル可） | 推奨 |
| F-DV-09 | グリッドビューへの復帰 | Escキーで即座にグリッドビューに戻る。戻った際に直前の表示位置を維持 | 必須 |

### 3.3 タグクラウド表示・編集（コア機能）

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|

## 5. 非機能要件

### 5.1 パフォーマンス

| 項目 | 要件 |
|------|------|
| 初期読み込み | 1,000枚の画像セットを10秒以内に読み込み完了 |
| サムネイル表示 | 仮想スクロール（Virtual Scrolling）によるスムーズな一覧表示 |
| タグ操作 | 即座に反映（100ms以内） |
| 自動保存 | 最後の操作から500ms後にデバウンス保存 |

### 5.2 データ安全性

| 項目 | 要件 |
|------|------|
| バックアップ | 初回読み込み時にタグファイルのバックアップを作成 |
| 排他制御 | 同一ファイルへの同時書き込み防止 |
| 文字エンコーディング | UTF-8の一貫した取り扱い |

### 5.3 ユーザビリティ

| 項目 | 要件 |
|------|------|
| レスポンシブ | 最低1280px幅からの対応 |
| アクセシビリティ | キーボード操作で主要機能が完結 |
| ダークモード | 長時間の画像確認作業を考慮し、ダークモード対応 |

---

## 6. 制約・前提条件

- ローカル環境での実行を基本とする（ファイルシステムへの直接アクセスが必要）
- 画像ファイルは読み取りのみ（このツールでは画像自体の編集は行わない）
- タグファイルの読み書きが主たるI/O
- 同時に扱うデータセットは1つ（複数データセットの同時編集は対象外）
- 対応ブラウザ: Chrome/Edge最新版
